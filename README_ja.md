## 🌟 主な機能

### マルチモデル・マルチプロバイダーサポート

- 🤖 OpenAI、Anthropic、Googleなどの主要AIプロバイダーをサポート
- 🔄 柔軟なモデル切り替えと設定、同じ会話内でいつでも異なるモデルに切り替え可能
- 🎛️ 高度な会話パラメータ制御(温度、top-p、トークン制限など)
- 📊 MCP(Model Context Protocol)サーバー統合

### Vibeモード(Vibe Coding)

- 🤖 Claude Code統合、自然言語での要件記述をサポート、AIが自動的に開発を完了
- ☁️ クラウドサンドボックス環境、Node.js、Python、Git、CMakeなどの開発ツールチェーンがプリインストール、ゼロ設定ですぐに使用可能
- 🔒 独立して分離されたクラウド実行環境、AI操作はローカルファイルに影響しない
- 📋 タスクパネルがAI実行プロセスをリアルタイムで表示、複数のAIタスクのバッチ管理と自動実行を実現
- 🚀 ワンクリックプロジェクトデプロイ、即座に起動、永続的なホスティング
- 🎭 コード実行結果の並列プレビュー、プレビュー、ファイルツリー、ターミナルの3つのビューモードをサポート
- 🧠 Planモードをサポート、AIが実装方法を計画してから実行、複雑なタスクやアーキテクチャ設計に適している

### Claude Skills

- 📦 4つの作成方法をサポート:手動作成、ファイルアップロード、GitHubインポート、履歴生成
- 🔧 17個の公式Skills内蔵、すぐに使用可能
- 📝 ビジュアル管理インターフェース、Skillsの編集と整理が簡単

### ドキュメント・データ処理

- 🖼️ 画像をアップロードしてAIにコンテンツ分析と説明生成を依頼
- 📄 複数のファイル形式をサポート
- 💻 コードシンタックスハイライト
- 📊 Mermaid図の可視化
- 📝 完全なMarkdownレンダリングサポート

### 優れたユーザー体験

- 🖥️ Windows、Mac、Linuxのマルチプラットフォームサポート
- 🌙 カスタマイズ可能なライト/ダークテーマシステム、カスタムCSSスタイルをサポート
- 👤 アカウントログインをサポート、残高と使用状況の照会が可能
- 📱 レスポンシブデザイン、様々な画面サイズに完璧に対応

### 効率的なワークフロー

- 🗂️ 複数の会話スレッドを同時に管理、思考が明確で混乱しない
- ⚡ リアルタイムストリーミングレスポンスをサポート
- ⌨️ 完全なキーボードショートカットシステム
- 🧰 302.AIツールマーケットプレイス内蔵、50以上のAIアプリケーションツールをカバー

### データ管理・プライバシー

- 📂 会話記録はローカルに保存、プライバシーを保護
- ☁️ クラウド同期をサポート、デバイス間でアクセス可能
- 🕶️ 会話はシークレットモードをサポート、チャット記録を保存しない
- 📤 会話履歴のインポート・エクスポートをサポート

### 多言語サポート

- 🇨🇳 中国語
- 🇺🇸 英語
- 🇯🇵 日本語(後続サポート)

## 🚀 クイックスタート

### 📋 システム要件

- **オペレーティングシステム**: Windows 10+ / macOS 10.14+ / Linux (Ubuntu 18.04+)
- **Node.js**: 18.x以上
- **パッケージマネージャー**: pnpm 10.18.3+(必須)
- **メモリ**: 4GB RAM(8GB以上推奨)
- **ストレージ**: 500MB以上の空き容量
- **ネットワーク**: 安定したインターネット接続(AIプロバイダーAPIにアクセスするため)

### ⚡ インストールと起動

```bash
# 1️⃣ プロジェクトをクローン
git clone https://github.com/302ai/302-AI-Studio.git
cd 302-AI-Studio

# 2️⃣ 依存関係をインストール
pnpm install

# 3️⃣ 開発サーバーを起動 🎉
pnpm dev
```

> [!WARNING]
> このプロジェクトは`pnpm`をパッケージマネージャーとして使用する必要があります。プロジェクトにはSvelteKitに必要なパッチが含まれており、他のパッケージマネージャーでは正常に動作しない場合があります。

## 📦 ビルドとデプロイ

### 🔧 開発コマンド

```bash

pnpm dev

# 型チェック
pnpm check

# コードリント
pnpm lint

# コードフォーマットチェック

pnpm quality

# すべての問題を自動修正
```

```bash
# ユニットテストを実行
pnpm test:unit

# E2Eテストを実行
pnpm test:e2e

# すべてのテストを実行
pnpm test
```

```bash

pnpm publish
```

## ✨ 302.AIについて

[302.AI](https://302.ai)は従量課金制のAIアプリケーションプラットフォームで、AIを実践に応用する際の最後の1マイルの問題を解決します。

1. 🧠 最新かつ最も完全なAI能力とブランドを集約、言語モデル、画像モデル、音声モデル、動画モデルなどを含むがこれらに限定されない
2. 🚀 基盤モデルに基づく深いアプリケーション開発、単純なチャットボットではなく真のAI製品を開発
3. 💰 月額料金ゼロ、すべての機能が従量課金制、完全にオープン、真に低い参入障壁と高い上限を実現
4. 🛠️ 強力な管理バックエンド、チームと中小企業向け、一人で管理、複数人で使用
5. 🔗 すべてのAI能力がAPIアクセスを提供、すべてのツールがオープンソースでカスタマイズ可能(進行中)
6. 💡 強力な開発チーム、週に2-3個の新しいアプリケーションをリリース、製品は毎日更新。参加に興味のある開発者も歓迎します
